A — Three Separate Tabs

Agents, MCP Servers, and Skills each get their own dedicated activity bar icon and sidebar. The current "Forge AI" tab is removed or repurposed (e.g. becomes a home/workspace view).

Pros

  • Direct access to any area
  • Clear mental model per domain
  • Expandable independently

Cons

  • More icons in activity bar
  • Workspaces panel needs a home

B — One Tab, Sub-Tabs Inside

Keep the single "Forge AI" activity bar icon, but add horizontal sub-tabs inside the sidebar for Agents, MCP Servers, Skills, and Workspaces.

Pros

  • One icon, less clutter
  • All Forge UI in one place

Cons

  • Not what VS Code expects
  • Harder to navigate quickly
  • Custom tab system to build
