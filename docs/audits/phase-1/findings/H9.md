## Scope

Area: `crates/forge-shell` (`tauri.conf.json`). Threat class: **T5 — CSP / HTML injection**. The webview renders LLM-originated text and tool-call outputs. CSP is the standard second-line defense against raw-HTML injection. With CSP disabled, any future raw-HTML sink, markdown-renderer-emitted script, compromised transitive dep, or dev-server mix-up becomes a full webview compromise — and because the Tauri capability file grants the same `core:default` to every window, a compromised webview has `invoke` access to every IPC command in the shell.

## Finding

- **Location:** `crates/forge-shell/tauri.conf.json:17` (`"csp": null`)
- **Severity:** high
- **Threat class:** T5

`app.security.csp` is explicitly `null`, disabling Tauri's auto-injected Content-Security-Policy. Current frontend code avoids every raw-HTML sink (verified by `web/packages/*` grep for inner-HTML assignments, dangerous set-HTML helpers, dynamic-code-eval patterns, markdown/sanitization libraries, legacy doc-writing APIs, adjacent-HTML insertion, and `foreignObject` — zero matches). But the threat posture depends on this being preserved forever by every contributor. A single raw-HTML sink slipped into `ChatPane` renders LLM output directly into the DOM.

With CSP in place, inline and dynamic-eval scripts are blocked and `connect-src` restricts outbound traffic — most practical injection gadgets stop functioning. Today there is nothing behind the first line.

`withGlobalTauri: false` is correctly set (no `window.__TAURI__` global), but it does not block `invoke` via `window.__TAURI_INTERNALS__`.

## Reproduction

1. Open `crates/forge-shell/tauri.conf.json` — observe `"security": { "csp": null }`.
2. Build and launch the Tauri shell.
3. DevTools → Network → the bootstrap document has no `Content-Security-Policy` header.
4. Any future raw-HTML assignment (e.g. direct assignment of LLM text to an element's inner-HTML property) executes without protection.

## Remediation

Set a concrete CSP in `tauri.conf.json`:

```json
"csp": "default-src 'self' ipc: http://ipc.localhost; script-src 'self'; style-src 'self' 'unsafe-inline'; img-src 'self' data: asset: https://asset.localhost; connect-src 'self' ipc: http://ipc.localhost; frame-ancestors 'none'; base-uri 'self'; object-src 'none'"
```

Tauri 2 auto-injects the nonce for its bootstrap when CSP is non-null. Exercise via the existing Playwright suite under `web/packages/app/tests/phase1/` to catch fallout.
